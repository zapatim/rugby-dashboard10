# Rugby Dashboard V5 — installation sur GitHub

Cette version comprend :
- MHR Espoirs en bleu foncé ;
- Crabos en bleu clair ;
- RCO Salagou en jaune ;
- le préfixe « J- » dans la couleur de l’équipe et le nombre en blanc ;
- une étiquette « CHAMPIONNAT » ou « AMICAL » sous le décompte ;
- aucune mention « M.A Match amical » sous Provence Rugby.

## Mise à jour du dépôt

1. Décompressez `Rugby_Dashboard_V5_GitHub.zip`.
2. Ouvrez le dépôt GitHub `rugby-dashboard10`.
3. Cliquez sur **Add file**, puis **Upload files**.
4. Glissez tous les fichiers contenus dans le dossier décompressé.
5. Acceptez le remplacement des anciens fichiers.
6. En bas de la page, cliquez sur **Commit changes**.
7. Attendez une à deux minutes.
8. Ouvrez le site avec `?v=5` à la fin de l’adresse.
9. Sur Mac, faites **⌘ + Maj + R** pour forcer l’actualisation.

Adresse :
`https://zapatim.github.io/rugby-dashboard10/?v=5`
