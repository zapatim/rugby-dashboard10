# Installation rapide

Cette version fonctionne avec tous les fichiers directement à la racine du dépôt.

1. Supprime les anciens fichiers du dépôt GitHub.
2. Importe tous les fichiers de ce ZIP.
3. Active GitHub Pages : Settings → Pages → Deploy from a branch → main → /(root).
4. Ouvre ensuite : https://zapatim.github.io/rugby-dashboard/

Pour le widget :
1. Installe Scriptable sur l’iPhone.
2. Ouvre `RugbyDashboard.scriptable.js` sur GitHub et copie tout.
3. Dans Scriptable, crée un script nommé Rugby Dashboard et colle le code.
4. Lance-le une fois.
5. Ajoute un grand widget Scriptable à l’écran d’accueil et sélectionne ce script.

Le widget choisit automatiquement le prochain match de chaque équipe.
