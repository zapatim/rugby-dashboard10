# Installation — Rugby Dashboard V4.1 pour Mac

Cette version corrige les logos très sombres, notamment Provence Rugby.

1. Décompressez le ZIP.
2. Dans le dépôt GitHub `rugby-dashboard10`, cliquez sur **Add file → Upload files**.
3. Glissez tous les fichiers du dossier décompressé.
4. Remplacez les fichiers existants et cliquez sur **Commit changes**.
5. Attendez environ une minute.
6. Ouvrez le dashboard avec `?v=4.1` à la fin de l’adresse, puis rechargez avec **⌘ + Maj + R**.

Les logos sombres répertoriés dans `DARK_LOGOS` sont automatiquement placés sur une tuile blanc cassé aux coins arrondis. Les autres logos restent inchangés.
